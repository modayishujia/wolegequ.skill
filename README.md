# wolegequ.skill

各赛道爆款起号技能，覆盖 16 个热门赛道的完整起号方法论。

## 这是什么

wolegequ.skill 是一个 AI 技能文件，帮助你在小红书、抖音、B站等平台快速起号。包含：

- 赛道深度分析与用户画像
- 3 秒留人技巧与爆款公式
- 口播稿与文字稿模板
- 关键词布局与 Hashtag 组合
- Day 1 行动清单
- 30 天冷启动计划
- 变现路径与报价公式

## 安装方式

直接告诉你的 AI：

```
安装这个 skill：https://github.com/modayishujia/wolegequ.skill
```

支持 Codex、MiMoCode、Claude Code、Cursor 等所有支持 skill 的 AI 工具。

## 使用方式

安装后，直接告诉 AI 你的需求：

### 🚀 起号指南

```
我想做健身运动赛道，请帮我起号
```

```
我要做美妆护肤，给我一份 30 天冷启动计划
```

### 🔥 爆款生成器

```
帮我生成 5 个健身赛道的爆款标题
```

```
给我写一篇小红书美食图文，要爆款模板
```

```
帮我写一个抖音短视频口播稿，主题是：减脂餐教程
```

### 📋 选题规划

```
帮我规划一周的科技赛道选题
```

```
给我 10 个育儿赛道的热门选题
```

### ✨ 内容优化

```
帮我优化这个标题：居家健身教程
```

```
这段口播稿太平淡了，帮我改成爆款节奏
```

### 💰 变现咨询

```
1 万粉的健身博主怎么变现？
```

```
美妆赛道品牌合作怎么报价？
```

AI 会自动加载对应赛道的参考文件，为你提供完整的起号指南、爆款模板和 Day 1 行动清单。
