# wolegequ.skill

各赛道爆款起号技能，覆盖 16 个热门赛道的完整起号方法论。

## 这是什么

wolegequ.skill 是一个 AI 技能文件，帮助你在小红书、抖音、B站等平台快速起号。包含：

- 赛道深度分析与用户画像
- 3 秒留人技巧与爆款公式
- 口播稿与文字稿模板
- 关键词布局与 Hashtag 组合
- Day 1 行动清单
- 30 天冷启动计划
- 变现路径与报价公式

## 使用方式

### 1. 选择赛道

告诉 AI 你想要起号的赛道：

```
我想做健身运动赛道，请帮我起号
```

或者：

```
我要做美妆护肤，给我一份起号指南
```

### 2. 获取起号指南

AI 会自动加载对应赛道的参考文件，为你提供：

- 赛道分析与用户画像
- 账号定位与人设搭建
- 内容策略与爆款公式
- 30 天冷启动计划

### 3. 生成爆款内容

让 AI 帮你生成：

```
帮我生成 5 个健身赛道的爆款标题
```

```
给我一个抖音短视频的口播稿模板
```

### 4. 执行 Day 1 行动

按照 AI 提供的 Day 1 行动清单开始执行：

1. 注册账号 + 完善资料
2. 发布第一条内容
3. 去热门博主评论区互动
